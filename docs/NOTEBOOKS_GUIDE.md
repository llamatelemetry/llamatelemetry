# Notebooks Guide

The `notebooks/` directory contains 18 curated notebooks with Kaggle-ready workflows and deep dives.

## Tracks

- Foundation (01-04): quickstart, server setup, multi-GPU, quantization
- Integration (05-10, 17): Unsloth, Graphistry, Kaggle large models, full workflow, W&B
- Advanced (11-13): GGUF visualizations and attention exploration
- Observability (14-16, 18): OpenTelemetry and production monitoring
