# LlamaTelemetry Kaggle Notebooks

This folder contains the current Kaggle notebook set for llamatelemetry v0.1.0.
Use the notebooks in order for a smooth learning path.

## Notebook Index
1. `01-quickstart-llamatelemetry-v0-1-0-e1.ipynb` - Quickstart: install, load model, run inference
2. `02-llama-server-setup-llamatelemetry-v0-1-0-e1.ipynb` - ServerManager setup + /health /props /slots /metrics
3. `03-multi-gpu-inference-llamatelemetry-v0-1-0-e1.ipynb` - Multi-GPU config (split mode, NCCL config)
4. `04-gguf-quantization-llamatelemetry-v0-1-0-e1.ipynb` - GGUF reports and suitability checks
5. `05-unsloth-integration-llamatelemetry-v0-1-0-e1.ipynb` - Unsloth export to GGUF (example)
6. `06-split-gpu-graphistry-llamatelemetry-v0-1-0-e1.ipynb` - Split GPU session + Graphistry basics
7. `07-knowledge-graph-extraction-graphistry-v0-1-0-e1.ipynb` - Knowledge graph builder example
8. `08-document-network-analysis-graphistry-e1.ipynb` - Document similarity network example
9. `09-large-models-kaggle-llamatelemetry-e3.ipynb` - Large model suitability + presets
10. `10-complete-workflow-llamatelemetry-v0-1-0-e1.ipynb` - End-to-end Kaggle pipeline helper usage
11. `11-gguf-neural-network-graphistry-vis-executed-e1.ipynb` - GGUF metadata + embedding kNN graph
12. `12-gguf-attention-mechanism-explorer-executed-e1.ipynb` - Attention matrix graph example
13. `13-gguf-token-embedding-visualizer-executed-e1.ipynb` - Token embedding kNN visualization
14. `14-opentelemetry-llm-observability-e5.ipynb` - OTLP setup + instrumented client
15. `15-rt-performance-monitoring-llamatelemetry-e3.ipynb` - PerformanceMonitor + /metrics parsing
16. `16-production-observability-llamatelemetry-e2.ipynb` - Production observability pipeline setup
17. `17-llamatelemetry-wandb-kaggle-notebook-e2.ipynb` - W&B logging + telemetry pipeline (updated)
18. `17-otel-graphistry-trace-glue-e2.ipynb` - Trace-to-graph helpers for Graphistry
19. `17-llamatelemetry-wandb-kaggle-notebook.ipynb` - Legacy W&B notebook (pre-e2)
